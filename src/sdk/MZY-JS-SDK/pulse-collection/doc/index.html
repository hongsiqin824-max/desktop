最小使用流程

  1. 创建 `PulseCollectionClient` 实例。
  2. 先注册事件，再请求串口权限。
  3. 调用 `requestPort()` 选择设备，调用 `open()` 打开串口。
  4. 调用 `start(mode, side)` 开始采集，完成后在 `onCollectionCompleted` 中保存结果。
如果你只想快速验证设备是否可用，先直接看 接入页。
