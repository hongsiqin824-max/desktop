设备连接 采集控制 设备控制 设备信息 返回格式

设备连接

方法说明
requestPort(filters?)弹出串口选择窗口，获取用户授权。
open(port?, options?)打开串口连接。
close()关闭串口连接。
isOpen获取当前连接状态。

采集控制

方法说明
start(mode, side)开始采集。mode 取值为 0 标准模式、1 快速模式；side 取值为 1 左手、2 右手。
stop()结束当前采集流程。
clear()清空当前与历史缓存。

设备控制

方法说明
setHandModel(side)设置左右手模式。
enterHandSearch()进入手动寻脉模式。
setMoveUp()探头上移。
setMoveDown()探头下移。
setMoveForward()探头前伸。
setMoveBack()探头回缩。
allowHandSearch()允许人工辅助寻脉。

设备信息

方法说明
getDeviceStatus()获取设备运行状态，结果通过 onDeviceStatus 回调。
readDeviceNo()读取设备编号，结果通过 onDeviceNo 回调。

异步返回格式

{
  code: number;
  data: any;
  message: string;
}
code说明
0成功
-1失败
