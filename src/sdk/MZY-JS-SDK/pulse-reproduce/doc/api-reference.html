初始化

const client = PulseReproduceClient.create(options);

create() 用于创建脉复仪客户端实例，可传入串口配置覆盖默认值。

设备连接

方法说明
requestPort(filters?)弹出串口选择窗口，获取用户授权并选择目标设备。
open(port?, options?)打开已选择的串口连接，连接成功后开始接收设备数据。
close()断开串口连接并释放资源。
getPorts()读取浏览器已授权过的串口设备列表。
isOpen只读属性，表示当前是否已连接。

复现控制

方法说明
sendData(dataList, coefficient, side)将脉象数据发送到设备并开始复现。
stop()停止当前复现。

sendData 参数

参数类型默认值说明
dataListnumber[][]必填脉象数据，9 组或 18 组。
coefficientnumber0.8输出力度系数，范围 0.5 到 1.2。
side0 | 10用户当前把脉手，0 表示右手，1 表示左手。

配置项

参数类型默认值说明
baudRatenumber115200串口波特率。
dataBits7 | 88数据位。
stopBits1 | 21停止位。
paritystringnone校验方式。
bufferSizenumber255缓冲区大小。

异步返回格式

除 requestPort()、getPorts() 这类浏览器串口对象方法外，主要异步控制方法返回统一结果。

{
  code: 0 | -1;
  message: string;
}
code说明
0成功
-1失败

导出内容

名称说明
PulseReproduceClient脉复仪 SDK 客户端。
REPRODUCE_DEFAULT_OPTIONS默认串口与 SDK 配置。
parseReproduceRealtimeFrame(frameBytes)解析设备实时返回帧，返回寸、关、尺数据。
