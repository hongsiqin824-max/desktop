Browser SDK / Web Serial API

让开发人员快速接入脉象复现 SDK

PulseReproduceClient 用于连接脉复仪设备，将脉象数据发送到设备进行复现，并接收设备实时返回的寸、关、尺数据。

开始接入 查看 API 查看事件
3 步复现 选择串口、打开设备、发送脉象数据。
3 个关键约束 浏览器需支持 Web Serial API，页面需运行在 HTTPS 或 localhost，串口授权必须由用户手动触发。
1 个协作入口 可直接使用脉诊仪采集结果中的 motorData 作为复现数据源。

最小使用流程

  1. 通过 PulseReproduceClient.create() 创建客户端。
  2. 先注册事件，再请求串口权限。
  3. 调用 requestPort() 选择设备，调用 open() 打开串口。
  4. 调用 sendData(dataList, coefficient, side) 发送脉象数据并开始复现。
如果你只想快速验证设备是否可用，先直接看 接入页。
