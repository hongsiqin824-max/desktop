最小使用流程

  1. 调用 PulseCollectionPenClient.create() 获取单例客户端。
  2. 注册实时数据、进度、完成和错误事件。
  3. 在用户点击操作中调用 requestDevice()，再调用 connect(device)。
  4. 调用 setPulsePosition('cun' | 'guan' | 'chi') 设置当前采集部位；设备上报后 SDK 自动收集。
  5. 必要时调用 enterCollect() 切换设备到采脉状态。
  6. 在 collection-completed 中保存浮中沉记录和分析结果。
