运行要求

  • 使用支持 Web Bluetooth API 的 Chrome 或 Edge。
  • 页面运行在 HTTPS 或 localhost。
  • 首次蓝牙授权必须由用户点击操作触发。
  • iOS Safari、Firefox 等不支持 Web Bluetooth 的环境无法连接。
const env = client.checkEnv();
if (!env.ok) {
  console.error(env.code, env.reason);
}

引入方式

import {
  PulseCollectionPenClient
} from '@mzy-devices-link/pulse-collection-pen';

const client = PulseCollectionPenClient.create();

浏览器打包产物也会挂载全局对象：

const client = window.PulseCollectionPenClient.create();

初始化参数

参数类型默认值说明
namePrefixstringMZY默认设备名称前缀。
filtersarray-自定义 Web Bluetooth 筛选条件。
optionalServicesarray-额外允许访问的蓝牙服务。
autoReconnectbooleanfalse意外断开后是否自动重连。
reconnectAttemptsnumber3自动重连最大次数。
reconnectDelayMsnumber1000重连间隔，单位毫秒。
verboseLogbooleanfalse是否在控制台输出 SDK 业务日志。
蓝牙协议参数和稳定采集规则均由 SDK 内部管理，使用层无需配置或维护。

推荐连接流程

const client = PulseCollectionPenClient.create({
  autoReconnect: true,
});

client
  .onConnected((device) => console.log('连接成功', device))
  .onRealtimeData((data) => console.log('实时数据', data))
  .onCollectionProgress((progress) => console.log('进度', progress))
  .onCollectionCompleted((result) => console.log('采集完成', result))
  .onCollectionInvalid((event) => console.warn('采集无效', event))
  .onError((error) => console.error(error));

const authorizedDevices = await client.getAuthorizedDevices();
const device =
  authorizedDevices.find((item) => item.gatt?.connected) ||
  authorizedDevices.find((item) => item.name?.startsWith('MZY')) ||
  await client.requestDevice();

const result = await client.connectAuthorizedDevice(device);
if (result.code !== 0) throw new Error(result.message);

设置寸、关、尺采集部位

const positionResult = client.setPulsePosition('guan');
if (positionResult.code !== 0) {
  console.error(positionResult.errorCode, positionResult.message);
  return;
}

await client.enterCollect();

设备开始上报后，SDK 会自动收集浮（1）、中（3）、沉（5）稳定数据，不需要额外调用 startCollection()。单档采集完成时触发 pressure-completed，三档完成时触发 collection-completed。

startCollection(position?) 仅用于主动清空上一轮缓存并重新设置部位，不会向设备发送开始采集命令。
