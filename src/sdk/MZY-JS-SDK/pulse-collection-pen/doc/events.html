注册方式

client
  .onStatusChange((event) => console.log(event))
  .onRealtimeData((data) => console.log(data))
  .onCollectionProgress((progress) => console.log(progress))
  .onPressureCompleted((result) => console.log(result))
  .onCollectionCompleted((result) => console.log(result))
  .onCollectionInvalid((event) => console.warn(event))
  .onError((error) => console.error(error));

client.on('device-id', (deviceId) => console.log(deviceId));

事件列表

事件快捷方法回调数据
status-changeonStatusChange连接状态、消息、设备信息。
device-selectedonDeviceSelected设备 ID 与名称。
connected / disconnectedonConnected / onDisconnected设备信息或 null。
reconnect-failedonReconnectFailed错误码和消息。
realtime-dataonRealtimeData实时压力和脉搏数据。
pressure-levelonPressureLevel0 | 1 | 2 | 3 | 4。
device-statusonDeviceStatus1 | 2 | 3。
collection-progressonCollectionProgress当前档进度。
pressure-completedonPressureCompleted单个压力档的完整采集结果。
collection-completedonCollectionCompleted三档记录和分析结果。
collection-invalidonCollectionInvalid本次稳定采集无效的原因。
device-idonDeviceId设备编号字符串。
log / error-messageonLog / onError日志字符串或错误对象。

实时数据与进度

// realtime-data
{
  position: 'cun' | 'guan' | 'chi' | null;
  pressureType: 1 | 3 | 5 | null;
  staticPressure: number;
  pulseValue: number;
  pressureLevel: 0 | 1 | 2 | 3 | 4;
  deviceStatus: 1 | 2 | 3;
  stable: boolean;
  timestamp: string;
}

// collection-progress
{
  position: 'cun' | 'guan' | 'chi' | null;
  pressureType: 1 | 3 | 5;
  percent: number;
}
稳定和不稳定数据都会触发实时事件。绘图可以节流，但 SDK 采集逻辑应完整接收所有数据。

完成事件

// pressure-completed
{
  position: 'cun' | 'guan' | 'chi' | null;
  pressureType: 1 | 3 | 5;
  records: PenRealtimeData[];
  timestamp: string;
}

// collection-completed
{
  mode: 1;
  records: PenProcessedPulseRecord[];
  analysis: PenPulseAnalysisReport;
  timestamp: string;
}

PenProcessedPulseRecord 包含部位、压力档、平均压力、原始业务数据、单周期结果、对齐后的单周期数据和脉复仪使用的 motorData。

无效采集

{
  position: 'cun' | 'guan' | 'chi' | null;
  pressureType: 1 | 3 | 5 | null;
  reason:
    | 'unstable-frame'
    | 'early-end'
    | 'new-start'
    | 'pressure-mismatch'
    | 'invalid-frame';
}
原因说明
unstable-frame稳定采集完成前恢复为不稳定数据。
early-end稳定阶段提前结束。
new-start当前档未完成时收到新的稳定开始信号。
pressure-mismatch上传压力档与当前采集档不一致。
invalid-frame协议数据不符合当前采集状态。
