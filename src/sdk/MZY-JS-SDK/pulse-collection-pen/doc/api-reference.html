设备连接

成员说明
create(options?)创建或返回全局单例客户端。
checkEnv()检查安全上下文和 Web Bluetooth 支持情况。
requestDevice(options?)打开浏览器蓝牙选择窗口并返回设备。
getAuthorizedDevices()获取浏览器已授权设备；不弹出选择窗口。
connectAuthorizedDevice(device, options?)直接连接已授权设备。
connect(device?, options?)建立 GATT 连接并初始化协议通道。
disconnect()停止通知、清理采集状态并断开连接。
reconnect()使用最近一次设备对象重新连接。
getDevice()返回当前设备对象或 null。
isConnectedGATT 是否已连接。
isProtocolReady写入与通知特征是否已就绪。
canExecute连接和协议通道是否均可用。

采集控制

方法说明
setPulsePosition('cun' | 'guan' | 'chi')设置寸、关、尺采集部位。设备上报后 SDK 自动开始收集。
startCollection(position?)可选：清空本轮缓存并设置采集部位；不会向设备发送开始采集命令。
clearCollection()清空接收缓冲、当前档和已完成记录。
analyzePulseData({ float, middle, deep })使用数值数组或实时记录手动执行脉象分析。

设备控制

方法说明
readDeviceId()读取设备编号，结果通过 device-id 事件返回。
queryDeviceStatus()查询设备工作状态。
enterStandby()切换到待机状态。
enterSearch()切换到寻脉状态。
enterCollect()切换到采脉状态。
toggleDeviceMode()发送设备模式切换命令。

分析与状态快照

const report = client.analyzePulseData({
  float: floatRecords,
  middle: middleRecords,
  deep: deepRecords,
});

const snapshot = client.getSnapshot();

getSnapshot() 返回连接状态、协议状态、设备信息、采集部位、压力等级、设备状态、当前进度、最近错误和公开配置。

返回格式与错误码

{
  code: 0 | -1;
  message: string;
  errorCode?: PenErrorCode;
}
错误码说明
ENV_NOT_SUPPORTED环境不支持 Web Bluetooth。
REQUEST_CANCELLED / REQUEST_FAILED用户取消或设备请求失败。
NO_DEVICE / NO_GATT没有设备或设备不支持 GATT。
NOT_CONNECTED / PROTOCOL_NOT_READY连接或协议通道尚未就绪。
INVALID_POSITION / COLLECTION_NOT_READY采集部位或采集前置条件无效。
CONNECT_FAILED / DISCONNECT_FAILED / RECONNECT_FAILED连接生命周期操作失败。
WRITE_FAILED设备命令写入失败。
